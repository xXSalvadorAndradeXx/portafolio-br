Cubfinder

QA Tester & Frontend Developer

Objetivo

Objetivo: Validar y mejorar la estabilidad del sistema de gestión de cubículos, garantizando una experiencia de usuario consistente y libre de errores críticos.

Proceso de trabajo

Proceso: Análisis de requerimientos, diseño de pruebas, implementación de soluciones frontend y ejecución de pruebas funcionales y de regresión.

Herramientas utilizadas
Android Studio Figma Azure HTML5 CSS3 JavaScript
Competencias desarrolladas
Pruebas funcionales y aseguramiento de calidad (QA) Diseño de interfaces en Figma Desarrollo frontend responsivo Validación de sistemas y detección de errores
Resultados
Se logró validar y mejorar la estabilidad del sistema de cubículos, asegurando una interfaz funcional, limpia y sin errores críticos.
Evidencias
El proyecto evidencia el trabajo conjunto entre diseño, desarrollo frontend y pruebas QA durante todo el ciclo de desarrollo.
