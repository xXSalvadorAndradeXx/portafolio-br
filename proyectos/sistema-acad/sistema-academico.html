Sistema Académico

QA (Quality Assurance)

Objetivo

Objetivo: Entregar una plataforma académica estable y escalable que permita la gestión de estudiantes y calificaciones con alta confiabilidad.

Proceso de trabajo

Proceso: Revisión de especificaciones, planificación de pruebas, ejecución de pruebas funcionales, reporte y verificación de correcciones, despliegue y monitoreo.

Tecnologías utilizadas
Vue.js Vite JavaScript HTML CSS PHP (Laravel) MySQL
Competencias técnicas
Apliqué pruebas funcionales Responsive Design Validación de interfaces (UI/UX) y detección de errores Garantizando el correcto funcionamiento del sistema
Resultados
Se obtuvo una aplicación estable, funcional y sin errores críticos, con una interfaz amigable y adaptable a distintos dispositivos.
Evidencias claras
Mi trabajo se evidencia en las pruebas realizadas, validación de funcionalidades y corrección de errores durante el desarrollo.
